## Installation

To install Dracula-Soft, you need a plugin manager.
In the example, bellow we are going to use vim-plug.

```vim
Plug 'maxmx03/dracula.nvim', { 'branch': 'vim9' }

colorscheme dracula
colorscheme dracula-soft
```

## Transparency

```vim
g:dracula_transparency = true
```
