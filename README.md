## Installation

To install Dracula, you need a plugin manager.
In the example, bellow we are going to use vim-plug.

```vim
Plug 'maxmx03/dracula.nvim', { 'branch': 'vim' }
colorscheme dracula
colorscheme dracula-soft
```

## Transparency

```vim
let g:dracula_transparency = 1
```
